# AbilitiesWindow
Тестовое задание для компании Brickworks Games.
Сетка способностей строится автоматически на основе конфигов способностей.
Подразумевается запуск только в 1920х1080.
Точка входа - класс с многозначительным названием EntryPoint.
Был использован UIExtensions package для отрисовки линий, соединяющих способности (стандартный LineRenderer не умеет в UI).
